# Cache Redis

A Simples API with Redis cache 

## Installation

Use for install dependencies:

```bash
dotnet build
```

## Running - dotnet

- dotnet ef database update

## Documentation

- For see documentation, with server running, open in browser the address http://localhost:5000/swagger
